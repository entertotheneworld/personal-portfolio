# Personal Portfolio 💻


## 1. Download ↪
```shell
$ git clone https://github.com/entertotheneworld/personal-portfolio
```


## 2. Installation ⚙️
```shell
$ yarn
or
$ npm i
```
⚠️ To verify the installation, you must have a node_modules folder in the root.


## 3. Lauch server 🚀
```shell
$ yarn start
or 
npm start
```
