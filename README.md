## Getting Started

#### Step 1:

At the main folder execute the following command in console to get the required dependencies:

```bash
  npm install
```
or
```bash
  yarn install
```

#### Step 2:

At the main folder execute the following command in console to run the development server:

```bash
  npm run dev
```
or
```bash
  yarn dev
```
